# Presence logo assets

- `presence-mark.svg`: primary standalone mark in #2F628F
- `presence-lockup.svg`: mark + outlined `Presence` wordmark, optimized for 32–40 px headers
- `favicon.svg`: white simplified mark on a #2F628F square; designed for 16, 24, and 32 px
- `presence-mark-white.svg`: white mark for dark backgrounds

All files use solid vector fills, transparent backgrounds where applicable, tight viewBoxes, and no live text, strokes, gradients, shadows, or raster effects.

Recommended minimum sizes:
- Standalone mark: 16 px high
- Header lockup: 32 px high
- Clear space: at least one vertical-stem width around the mark
